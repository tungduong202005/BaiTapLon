Ngôn ngữ lập trình C++

C++ là một ngôn ngữ lập trình đa năng, được phát triển bởi Bjarne Stroustrup vào những năm 1980. C++ mở rộng từ C với các tính năng hướng đối tượng.

Cấu trúc cơ bản của một chương trình C++

using namespace std;
